jQuery Hysteresis plugin

Hysteresis adds a lag to controls to give the user more control over what happens on the screen. This little plugin manages timings when you process interface events.

Example usage: 

$(".menu > ul > li").hysteresis({
    show: function(el, e) {
        $(el).addClass("over");
    },
    hide: function(el, e) {
        $(el).removeClass("over");
    },
    inDelay: 200,
    outDelay: 300
});
$(".menu > ul > li").hover(function (e) {
    $(this).hysteresis("show", e);
}, function (e) {
    $(this).hysteresis("hide", e);
});
